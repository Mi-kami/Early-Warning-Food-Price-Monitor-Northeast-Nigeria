🔧 Source Code Directory (src)

This folder contains reusable Python scripts used throughout the project.

Rather than embedding all logic inside notebooks, core functionality such as:

- Data cleaning functions

- Feature engineering utilities

- Model evaluation helpers

- Configuration settings

are stored as modular .py files in this directory.

This structure promotes:

- Code reusability

- Maintainability

- Clarity

- Industry-level project organization

Notebooks import functions from this directory to maintain clean and scalable workflows.
